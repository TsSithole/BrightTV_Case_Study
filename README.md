Bright TV Analysis

Project Overview

Bright TV’s CEO aims to grow the subscription base this financial year. This project delivers data-driven insights to the Customer Value Management (CVM) team to help achieve that goal. The analysis focuses on:

Objective
- Understanding user profiles and consumption patterns.
- Identifying factors that influence viewing behavior.
- Recommending content for low consumption days.
- Proposing initiatives to accelerate user base growth.

Tools 
Planning & Architecture: Miro 
Data Processing: Databricks, SQL
Data Analysis & Visualization: Microsoft Excel, Pivot Tables, Charts 
Presentation: Microsoft PowerPoint

Task 1: Data Cleaning & Time zone Conversion

- Convert all UTC timestamps to SA time (UTC+2).
- Handle missing values, duplicate sessions, and outliers.

Task 2: Exploratory Data Analysis (EDA)
- User segmentation by location, age, race etc.  
- Consumption trends by hour of day, day of week, and seasonality.

Task 3: Factor Analysis
- Identify key drivers of consumption:
- Time of day and day of week effects.


Task 4: Low Consumption Day Recommendations
- Pinpoint days (e.g., mid week, post holiday) with below average viewing.
- Propose content bundles, reminders, or limited time events to boost engagement.

Task 5: Presentation and Recommendations 
- PowerPoint Presentation.
- Design targeted campaigns (e.g., trial extensions, family plans, localized content).
